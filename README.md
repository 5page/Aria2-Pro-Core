## Usage

### Install

* Automatic script:
```shell
curl -fsSL git.io/aria2c.sh | bash
```

* Manual installation:
```shell
tar zxvf aria2-[version]-static-linux-[arch].tar.gz
sudo mv aria2c /usr/local/bin
```

### Uninstall

```
sudo rm -f /usr/local/bin/aria2c
```
